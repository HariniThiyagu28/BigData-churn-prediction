# BigData-churn-prediction
Spark ML daily batch job for churn prediction 

## Project Overview
This project simulates a **daily batch job** that loads a trained Spark ML pipeline
and generates churn probability predictions for new monthly user data.

## Key Features
- Spark ML Pipeline (Random Forest)
- Batch inference on new user data
- Churn probability extraction from ML vectors
- CSV output for downstream systems
- Simulates enterprise daily batch processing

## Tech Stack
- Apache Spark (PySpark)
- Spark MLlib
- Google Colab
- GitHub
